# Store Screenshots

Raycast renders PNG files in this folder on the extension store page.

The current screenshots cover both provider families:

- `ai-voice-studio-minimax-*.png`: MiniMax long-form reading, voice selection, and clone-oriented workflow.
- `ai-voice-studio-mimo-*.png`: MiMo expressive TTS, TTS Studio, and voice-control workflow.

All screenshots are PNG files at 2000 x 1250 pixels.
